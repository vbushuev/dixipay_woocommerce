DixiPay Wocommerce Payment Gateway Plugin
-------------------------------------------------

Unzip and copy file dixipay.php in your wordpress root folder.

copy dixipay-payment-gateway-woocommerce folder in your wp-content/plugins folder.

Login To admin and install plugin .

Go To WooCommerce->Settings and click on Checkout tab.

Click on DIXIPAY from the checkout options showing at top.

Add your Client Key And Password .

Use 'https://secure.test.dixipay.eu/payment/auth' as Gateway Url.

Call Back
http://bufori.websitewelcome.com/~devproj/dixipay_wordpress/dixipay.php 


Save Your Changes . You are Good TO Go!
